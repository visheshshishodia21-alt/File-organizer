# 📂 File Organizer

A Python project that automatically organizes files into folders based on their extensions.

## 🚀 Features

- Organizes Images
- Organizes PDFs
- Organizes Documents
- Organizes Music
- Organizes Videos
- Organizes Python Files
- Automatically creates folders

## 🛠 Technologies

- Python
- os
- shutil

## ▶️ How to Run

```bash
python organizer.py
```

## 📂 Supported File Types

- Images (.jpg, .jpeg, .png)
- PDFs (.pdf)
- Documents (.docx, .xlsx, .pptx)
- Music (.mp3)
- Videos (.mp4)
- Python (.py)
- Web Pages (.mht)
